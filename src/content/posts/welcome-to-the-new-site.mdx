After a few years on Wix, I've moved this blog over to a site I built myself with Next.js. A few reasons for the switch:

- **Speed** — pages load faster since everything is statically generated.
- **Control** — I write posts as plain Markdown files instead of fighting a drag-and-drop editor.
- **Ownership** — the whole site lives in a git repo I control, not locked into a platform.

The design is intentionally simple for now. More posts, and more polish, coming soon.
